# Installation
Setup project:
```
npm install
```

Create `.env` file in the root directory using `.env.template`. `HOST` defines domain and port, the rest are endpoints.

# Commands available
Start development server: `npm start`
Build: `npm run build`
Lint: `npm run lint`
Format: `npm run format`
